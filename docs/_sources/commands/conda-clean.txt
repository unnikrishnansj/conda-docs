   NAME
          conda - conda clean
   
   DESCRIPTION
          usage:  conda  clean [-h] [-y] [--dry-run] [--json] [-i] [-l] [-t] [-p]
          [-s]
   
          Remove unused packages and caches.
   
   OPTIONS
      optional arguments:
          -h, --help
                 Show this help message and exit.
   
          -y, --yes
                 Do not ask for confirmation.
   
          --dry-run
                 Only display what would have been done.
   
          --json Report all output as json. Suitable for using conda programmati‚Äê
                 cally.
   
          -i, --index-cache
                 Remove index cache.
   
          -l, --lock
                 Remove all conda lock files.
   
          -t, --tarballs
                 Remove cached package tarballs.
   
          -p, --packages
                 Remove  unused cached packages. Warning: this does not check for
                 symlinked packages.
   
          -s, --source-cache
                 Remove files from the source cache of conda build.
   
   EXAMPLES
                 conda clean --tarballs
